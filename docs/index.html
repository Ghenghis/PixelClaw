// Capabilities

Core Features

Built for real development workflows with failsafes, retry logic, and backup plans at every layer.

💬
Inter-Agent Chat
Agents communicate through a message bus with threading, priority routing, broadcast, and direct messaging. Conversations are persisted in shared memory.
🔄
Seamless Model Switching
Switch any agent to a different LM Studio model mid-conversation. Context is summarized, stored in memory, and replayed to the new model automatically.
🧠
Shared Memory Store
Persistent JSON-backed memory for conversation summaries, project state, task results, and agent knowledge. Atomic writes prevent corruption.
🛡
Retry & Failsafes
Exponential backoff retry, circuit breakers, fallback models per role, health checks, and automatic error recovery keep agents running.
🎮
Pixel Art Characters
Animated pixel-art agents live inside VS Code. They react to real agent activity — coding, thinking, chatting, switching models — in real time.
🖥
52 Model Registry
Pre-configured model registry with 52 LLMs organized by role: code, reasoning, vision, tool use, chat, and Evony specialization. VRAM-aware scheduling.
